Electronic Design

On the third and fourth week, Mr Chew taught us how to design our own circuit using EAGLE software and how to be able to convert the schematic design into a PCB design and then mill out the design.

We will be following the ATTiny-412 hellow world board schematic but we will add a LED and switch to the schematic

What is EAGLE?

EAGLE is a software where you can create your own circuits.

Let's Get Started: Adding Libraries

Fab Library

  1. Click this link to bring you to gitlab fab library
  2. Click onto eagle > fab.lbr
  3. Then download the fab.lbr file
  4. Put the file into your file explorer under the folder EAGLE > libraries

Adafruit (Eagle Library)

  1. Click this link to bring you to github of adafruit library
  2. Click onto the arrow next to code and download the zip folder
  3. Then unzip the folder and put it into your file explorer under the folder EAGLE > libraries

SparkFun (Eagle Library)

  1. Click this link to bring you to github of SparkFun library
  2. Click onto the arrow next to code and download the zip folder
  3. Then unzip the folder and put it into your file explorer under the folder EAGLE > libraries

Pinhead+ Library

  1. Click this link to bring you to the website

  3. Then unzip the folder and put it into your file explorer under the folder EAGLE > libraries

How to use the library

Open your Eagles software and under libraries open the folder and right click the library you want to use and select use

Creating a New Project

  1. Right click new project > Type the name of the project
  2. Right click the new project you just named and go to new > schematic
  3. To change your cursor and background go to options > user interface > cursor: large and background: black or up to you to change the layout and the schematic
  4. Next go to grid > display: on > style:lines and the rest of the settings as shown
  5. Now you want to add the frame so go to add parts > type in the search asterisk frame asterisk and look for A4L-LOC
  6. Using the origin plus sign, place your frame at the corner of the origin, now you can do your schematic in the frame
  7. Following the schematic above find the components as follows, make sure the components has footprints

    8. Tiny412 Chip

    Search: ∗tiny412∗
    Name: ATTINY412-SSN

    Resistor x2

    Find the folder fab > R > R1206FAB

    Switch

    Find the folder fab > SW_SWITCH_TACTILE_6MM

    Capacitor Unpolarized

    Find the folder fab > CAP_UNPOLARIZED > CAP_UNPOLARIZEDFAB

    FTDI Connector

    Find the folder fab > CONN_06_FTDI-SMD-HEADER

    UPDI Connector

    Find the folder pinhead+ > PINHD-1X3 > PINHD - 1X3

    VCC x3

    Search: vcc
    Folder: Supply1 > VCC

    GND x4

    Search: gnd
    Folder: Supply1 > GND

  8. Once all the components are there, you can start wiring up your connections using the net icon. Leave a few nets unconnected which we will later label them for the RX and TX
  9. Using Name icon, to name the unconnected labels to connect them tgt by naming them RX and TX respectively. You can also go and put the values for the resistor and capacitor.

  11. To check, go to tools > ERC to check your circuit if your circuit has 0 errors then you are good to go, the warning you see on this screen is because I havent added any text to some of the components and the frame.
  12. To add text to frame use the text icon

Converting from Schematic to PCB

  1. Click the generate or switch to board icon to make the schematic circuit a pcb board
  2. Set your grid settings to display: on and style: lines
  3. Put all your components into the yellow box
  4. Move around your components until you are satisfied and making sure when you add the tracks later that no tracks should be overlapping one another
  5. Once done, move the circuit the the bottom left corner with the white plus sign origin and make the yellow box smaller making sure you leave a 2 box gap around the circuit.
  6. Once done let's set the design rules

    7. Clearance: 16mil for Wire, Pad, Via
    Clearance: 12mil for Smd

    Sizes Minimum Width: 15mil

  7. once done using the route airware to route the connections but first adjust the route settings above
  8. Then route your connections using 45 degree angle
  9. Once you are done, go to tools > DRC > check to check if there is any design errors, if no pop up window comes up means there is no errors
  10. To make the outline, using the line tool adjust to the settings first before drawing the outline along the yellow line
  11. Using the milter tool, you can make rounded corners or straight using the following setting of R100 (Radius 100)
  12. To remove the text labeling go options > set > misc > turn off display pad names and signal names on the pads and signal names on traces
  13. Next go to layer settings and hide all the layers and turn on only the top and pad layers, it should show only the traces and pads
  14. After you can export the traces and pads going to file > export > image, you change the resolution to 1000 and give it a file name and monochrome.
  15. Go back to layer settings, hide all layers and turn on only miling layer
  16. Then export the outline doing the same steps as you did in Step 14
  17. You can turn on the following layers after you have exported the traces and outline
  18. You can check if your circuit is ok the outlines and traces in nc viewer website

Editing the Finished PCB board using GIMP

You can use GIMP to edit your exported traces & outline images. For this case we want to edit one of the jumpers to be a surface mount type instead of the through hole type that it is now.

  1. Open up GIMP software and open the traces file first file > open
  2. We need to change the through hole components to surface mount components
  3. Using the magic wand tool, shift + select the 3 black holes
  4. Then go to edit > Fill with BG Color to invert the black hole color to white
  5. Then go to select > none to deselect the holes
  6. Now you can go to file to either export as a new png file or override the existing file
  7. Now open the outline file
  8. We now want to invert the inner black hole highlighted in green to white and the white outline around it to be black as black is the one that we want cut away and white is the parts we want to keep
  9. So doing the same steps as step 3 to 5 invert the inner black hole first
  10. Then go to select > invert
  11. Then go back to edit > fill with FG Color to fill the white outline to be black
  12. You can now export or overwrite the existing outline file and you are now ready to put it into mods website to prep for the files for miling
  13. Refer to my electronic production to know more about how to use mods to prepare the file
