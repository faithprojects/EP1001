Fusion360 Design & Manufacturing of a Star Mold

Fusion360 Design

  1. Make a box sketch of 70 by 70mm
  2. Using the Extrude tool, Extrude the sketch to 30mm
  3. Create a new sketch top box and offset by 5mm to create a wall for the mold
  4. Using the Extrude tool, Extrude down the sketch by 25mm
  5. Create a new sketch to start sketching my star, I used this video to help me sketched out the star
  6. Using the Extrude tool, Extruded the star.
  7. This is how my star mold looks like:

Fusion360 Manufacturing

  1. First change the workspace to manufacturing
  2. Then we want to setup the manufacturing process, so click on the setup icon
  3. We want to make sure that the arrows are in the middle of the box and in Work Coordinate System (WCS) the orientation should look like this:
  4. For the Stock, make sure the mode is Relative size box and then u can click ok now ur mold is set
  5. Next, click on to adaptive clearing and under tool tab we will be using a 3mm flat endmill and follow settings for Coolant, Spindle speed and cutting feedrate in the image below:
  6. For Geometry tab, tick the Stock Contours and select the inner line of the box
  7. For Heights tab, I left the clearance, retract and top height as the default but changed the bottom height as I left 5mm at the bottom
  8. For Passes, follow the settings for Max roughing Stepdown, Fine Stepdown and tick the box for stock to leave and follow the settings for that too:
  9. For linking I did not change any of the settings so we can leave it as it is and click ok and it will start to generate the cams and wait until it is done
  10. Next is to do the finishing pass, so once the first adapative clearing process that u just did is done. click onto the adapative clearing icon again
  11. This time the settings for the tool tab is the same as in step 5
  12. But for Geometry tab, the setting is the same as in step 6 but this time we tick the rest machining too
  13. For heights tab, we leave the settings as the default
  14. For Passes tab, we will untick the stock to leave box and change the Max roughing stepdown and fine stepdown
  15. For linking we can leave the settings as it is then click ok and wait for the cams to generate
  16. Once all the cams are done generating, you can simulate by right clicking the setup to simulate the whole process
  17. Once done we can now post process the manufacting process but before that we need to install the post processor for the stepcraft CNC in fusion 360.
  18. Click this post processor link and search Stepcraft UCCNC that will bring you straight to download the post processor for stepcraft CNC. This other instructions link shows you the steps to install the post processor too.
  19. After you have installed the post processor into Fusion360, you can right click the setup and click post process
  20. Then follow the settings for the post process and then click post in order to get a nc file of your mold design

Miling Process

I first cut my styrofoam stock with a 50mm thickness & a 60mm by 60mm square using pen knife. Using double sided tape, I taped the bottom of the stock and use it to stick on to the sacrificial board of the milling machine.

Then I marked the center point of the stock.

Then using the stepcraft software, setting my X and Y axis to the center of the styrofoam stock and then setting my Z axis. Im using a 3mm end mill bit.

My first try was unsuccessful as the stock size that I cut was too small which made the sides of the wall too thin.

So my second try, I cut the stock bigger 100mm by 100mm and this time it worked better than the first try

But it started to make a loud screeching sound because the 3mm end mill bit was not long enough so I had to cut some of the top part of the stryofoam but it ended up not coming out right.

So on my third try I cut some of the top part of the stryofoam before milling it but while milling it still needed to be cut away so I cut diagonally as it was much easier than cutting it straight.

Silicon Molding

For the silicon Molding, I will be using the Silicon rubber RTV-620. It has 2 parts A and B. It is important for us to read the datasheet to know the mix ratio, pot life and cure time of the silicon. Make sure to also prepare the table by using plastic sheets to cover up the table as it can get abit messy and remember to wear gloves.

First I pour Part A first into a cup and using the measuring scale to measure the amount I needed.

Next, I poured Part B into a different cup and using the same measuring scale to measure the amount needed. Make sure that both Part A and Part B are of equal amounts.

Then I have to combine Part B into Part A so that I can mix it together. Then I slowly mix the mixture as you don't want to mix too fast and let air bubbles in but I have to make sure that mix it slowly and making sure the mixture is well combined but also keeping track of how much pot life time I have.

Next, it was time to start pouring the mixture onto my styrofoam mold. I start pouring from the corner and pouring it slowly and letting the mixture flow into the mold itself.

Here is my finished Silicon pour and now we have to let it cure.

The following week, it was done curing and this is how it looked like:

I then proceeded to take it of the styrofoam mold but it was hard to take out as I did not spray on mold release before pouring the silicon which I had no choice but to break the styrofoam mold but the Silicon mold turned out great!

Resin Casting

For the resin casting, I will be using Smooth Cast 305. Same thing with the silicon, we have to take note of the mix ratio, pot life and cure time for the resin. Make sure the table is well prepared and wear rubber gloves.

First I pour Part A first into a silicon cup and using the silicon cup with volume measurements on it to measure the amount I needed.

Next, I poured Part B into a different silicon cup and measure the amount needed. Make sure that both Part A and Part B are of equal amounts.

Then I have to combine Part B into Part A so that I can mix it together. Then I slowly mix the mixture making sure not to mix too fast and let air bubbles in and making sure the mixture is well combined but also keeping track of how much pot life time I have.

Next, it was time to start pouring the mixture onto my silicon mold. I start pouring from the corner and pouring it slowly and letting the mixture flow into the mold itself.

Here is my finished resin pour and now we have to let it cure.

The following week the resin mold did not turn out right which was because the resin I used was expired:

So I then decided to switch it up and use a different resin the Polytek Easy Flo 90 Liquid plastic. Here is the datasheet for the Polytek:

During the pot life time of stirring the resin, once I was done stirring I poured the mixture into my silicon mold and minutes later the mixture became very warm to hold in the silicon cup I used.

After leaving it to cure over a week, I went to check my mold and this time it looked alot better than the first time other than the overflow that you can see but I could call this project done!
